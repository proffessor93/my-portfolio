# Портфолио — Dimon

Персональный сайт-портфолио начинающего frontend-разработчика. Живая версия: **[proffessor93.github.io/my-portfolio](https://proffessor93.github.io/my-portfolio/)**

Тёмная «терминальная» эстетика с янтарными акцентами, анимированный фон и встроенный **AI-ассистент** — живой чат с нейросетью прямо на странице.

## Что внутри

- **Чистые HTML / CSS / JavaScript** — без фреймворков и сборщиков, один файл
- **Анимации**: печатающаяся строка терминала, появление секций при скролле, подсветка карточек за курсором, живой фон из дрейфующих точек (canvas)
- **Адаптивная вёрстка** — корректно выглядит на телефоне
- **Доступность**: поддержка `prefers-reduced-motion`, focus-стили, aria-атрибуты
- **AI-чат виджет**: фронтенд на ванильном JS, общается с собственным бэкендом

## AI-ассистент (фулстек-часть)

Кнопка в углу открывает чат с нейросетью. Архитектура:

```
GitHub Pages (этот сайт)
        │  HTTPS (fetch)
        ▼
VPS: nginx (reverse proxy, TLS Let's Encrypt)
        ▼
FastAPI-бэкенд (systemd-сервис)
  · ключ LLM-провайдера хранится в .env на сервере
  · rate-limit по IP (защита от переспама)
  · CORS только для домена портфолио
        ▼
Groq API (LLM)
```

API-ключ никогда не попадает в браузер — фронтенд общается только с собственным бэкендом.
